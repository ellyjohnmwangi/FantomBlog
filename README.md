# FantomBlog

FantomBlog is a weblog where bloggers can write a post(article) which includes text and pictures and pulish it for all to see with the latest posts appearing first at the top. It is a platform where a writer or a group of writers can share their views on an individual subject.It is developed using python -Django Rest framework. 
## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install conda virtual environment and django.

```bash
pip install conda
pip install django
```

## Usage
create a conda virtual environmeant. Replace [virtualname] with your environnment name.
```bash
conda create virtualname
```
activate the virtual environment
```bash
conda activate virtualname
```
navigate to the file location of the file manage.py
```bash
cd filepath/Fantom
```
run local server via the command prompt 
```bash
python manage runserver
```
click on the https link to open the system in your browser.
